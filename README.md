## 📥 Installation

1. Download the **`practice.cfg`** file.
2. Place it in the following folder:

```
Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg\practice.cfg
```

3. Launch the game and open the console.
4. Run the command:

```
exec practice
```
